Detailed Description

Uses the candidate location list created by FastColorFilter and improved by TwoColorFilter to locate and validate CC2 (Color Circles type 2) markers. After localization and validation, it exports the detection results to a DetectionResultExporterBase.

To use it,

  • call init() and provide the name of the config file
  • set ResultExporter. Results will be exported there.
  • optionally, set verboseImage to see verbose information
  • for every frame, call LocateMarkers
